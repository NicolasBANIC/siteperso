## 🎬 Micro-interactions

### Hover sur Bouton Primaire
```
État normal :
┌──────────────────┐
│ Demander un devis│  scale(1.0)
└──────────────────┘

État hover :
┌────────────────────┐
│ Demander un devis  │  scale(1.05) + shimmer effect
└────────────────────┘
  ↓ Transition 200ms
```

### Pulsation du Badge
```
Frame 1:  🟢 ●  (opacity 1.0, scale 1.0)
Frame 2:  🟢 ◉  (opacity 0.7, scale 1.2)
Frame 3:  🟢 ◎  (opacity 0.5, scale 1.4)
Frame 4:  🟢 ○  (opacity 0.3, scale 1.6)
Frame 5:  🟢 ○  (opacity 0.0, scale 2.0)
  ↓ Loop infini
```

### Parallax au Scroll
```
Position initiale (scroll = 0):
┌─────────────────────────────────┐
│ [Image position Y: 0]            │
│ [Texte position Y: 0]            │
└─────────────────────────────────┘

Après scroll de 100px:
┌─────────────────────────────────┐
│ [Image position Y: 50px]         │  ← Vitesse 0.5x
│ [Texte position Y: 100px]        │  ← Vitesse 1x
└─────────────────────────────────┘
  = Effet de profondeur
```

## 📊 Performance Optimizations

### Lazy Loading
```
Viewport visible:
┌─────────────────────────────────┐
│ [Hero Section - Chargée]         │  ← priority={true}
└─────────────────────────────────┘
        ▼ Scroll ▼
┌─────────────────────────────────┐
│ [Section suivante - Lazy]        │  ← loading="lazy"
└─────────────────────────────────┘
```

### GPU Acceleration
```
Propriétés optimisées:
✅ transform: translateY()  ← GPU
✅ opacity                  ← GPU
❌ top/left                 ← CPU (évité)
❌ width/height             ← CPU (évité)
```
