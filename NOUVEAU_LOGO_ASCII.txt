                        📋 SPÉCIFICATIONS

═══════════════════════════════════════════════════════════════════════

🎨 COULEURS :
   • "BAN" + chevron gauche (<)  : Doré #FFD74A
   • "DEV" + chevron droit (>)   : Vert Matrix #4CFF73
   • Ligne sous BAN              : Doré #FFD74A
   • Lignes sous DEV             : Vert Matrix #4CFF73

📐 STRUCTURE :
   
   <  B A N   D E V  >
      ━━━━━━  ━ ━ ━━━━━━

   • Chevron gauche doré
   • Texte "BAN" en doré
   • Texte "DEV" en vert
   • Chevron droit vert
   • Ligne continue sous BAN
   • 3 petits traits + ligne longue sous DEV

✨ STYLE :
   • Format : SVG vectoriel
   • Fond : Transparent
   • Police : Sans-serif bold géométrique (Montserrat ExtraBold)
   • Effets : Aucun (pas de glow, pas d'ombre)
   • Alignement : Horizontal parfait

📏 DIMENSIONS :
   • ViewBox : 240 × 70 pixels
   • Hauteur mobile : 32px
   • Hauteur desktop : 40px
   • Scalable : 24px à 128px

                        📊 COMPARAISON

═══════════════════════════════════════════════════════════════════════

                    AVANT              →        APRÈS
                    ─────                       ─────
Format :            PNG/WebP/AVIF      →        SVG
Fichiers :          4 fichiers         →        1 fichier
Poids :             ~80 KB             →        ~2 KB
Scalabilité :       Limitée            →        Infinie
Identité :          Basique            →        Forte (bicolore)
Maintenance :       Complexe           →        Simple
