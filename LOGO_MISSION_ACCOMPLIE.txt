🎨 CARACTÉRISTIQUES DU LOGO

Structure :
  < BAN DEV >
    ━━━━━ ━ ━ ━━━━━

Couleurs :
  • BAN + < + ligne gauche    → Or métallique profond
  • DEV + > + traits/ligne    → Vert matrix sombre

Format :
  • Type : SVG vectoriel
  • ViewBox : 400×100
  • Fond : Transparent
  • Poids : ~1.5 KB
  • Police : Montserrat ExtraBold (800)

Effets :
  • Dégradés linéaires métalliques
  • Filtre d'ombre portée subtile
  • Lignes décoratives (2.5px, bouts arrondis)
