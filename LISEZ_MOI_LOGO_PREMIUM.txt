🎨 CARACTÉRISTIQUES DU LOGO

Structure :  < BAN DEV >
             ━━━━━ ━ ━ ━━━━━

Couleurs :
  • BAN + < + ligne gauche    → Or métallique profond (#D4AF37 → #B8860B)
  • DEV + > + traits/ligne    → Vert matrix sombre (#0B8F2D → #145A32)

Format :     SVG vectoriel transparent
Poids :      ~1.5 KB (ultra-léger)
Police :     Montserrat ExtraBold (800)
Effets :     Dégradés métalliques + filtre subtil
