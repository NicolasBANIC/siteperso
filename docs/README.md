## ⚡ Commandes Rapides

### Tests Automatisés

```bash
# Build production
npm run build

# Lint
npm run lint

# Format check
npm run format:check

# Lighthouse (si configuré)
npm run lighthouse
```

### Déploiement

```bash
# Vérifier que tout est OK
npm run lint && npm run build

# Déployer (Vercel)
vercel --prod

# Déployer (Netlify)
netlify deploy --prod
```
