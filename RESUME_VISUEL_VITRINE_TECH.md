## 🎯 ANIMATIONS PAR TYPE

### Scroll Reveal
```
Direction LEFT  : ←─────  (services, process odd)
Direction RIGHT : ─────→  (process even)
Direction UP    : ↑       (titles, CTA)
Direction FADE  : ○ → ●   (stats, badges)
```

### Hover Effects
```
Cards     : shimmer + glow + rotate(1deg) + scale(1.02)
Buttons   : shimmer + glow + scale(1.05)
Badges    : scale(1.1) + color change
Icons     : scale(1.1) + color change
Images    : scale(1.05) + brightness(1.1)
```

### Continuous Animations
```
Matrix Rain    : 60fps, falling characters
Particles      : 60fps, mouse interaction
MeshGradient   : Sinusoidal movement
Globe 3D       : Rotation 0.002 rad/frame
Code Typing    : 50ms/char, loop
Dashboard      : Update every 2s
```
